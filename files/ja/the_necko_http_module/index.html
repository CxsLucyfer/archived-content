技術ノート

クラスの概要 
このドキュメントでは、HTTP モジュールの実装に利用されるクラスの概要を説明します。HTTP 実装を拡張したいと考える開発者のためのガイドの役割を果たします。
トランザクションモデル 
このドキュメントでは、HTTP トランザクションの一生と、関連するスレッドやオブジェクトについて説明します。
パイプライン化 FAQ 
このドキュメントでは、HTTP/1.1 パイプライン化の概要を、いくつかのよくある質問に答える形で説明します。
キャッシュ FAQ 
このドキュメントでは、HTTP キャッシュがどのように Necko に実装されているのか、その概要を説明します。何がどのくらいの期間キャッシュされるのかといった、いくつかのよくある質問に答えます。
HTTP ロギング 
このドキュメントは、各種プラットフォームで HTTP のログ取得を有効にする方法を説明します。
