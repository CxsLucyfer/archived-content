ラッパー/バインディング

C++

  • Cocos2d-JS - Cocos2d-JS は SpiderMonkey を組み込んだクロスプラットフォームのゲームエンジンで、Web とネイティブプラットフォームの両方に同じ API を提供します。Getting started here.
  • Flusspferd - (新しい) C++ バインディング
  • libjspp - SpiderMonkey を拡張して埋め込むための C++ テンプレートベースのライブラリ。SpiderMonkey 1.8.5 以降で動作します。
  • Spiderape - SpiderMonkey の最も古い C++ バインディング
  • TriXUL - (TriXUL CVS) - Trixul XML ベースの GUI ツールキットは、SpiderMonkey を組み込み、JavaScript を使用してその GUI の背後にロジックを実装し、JavaScript から C++ オブジェクトへの呼び出しをサポートします。

Rust

  • mozjs - Servo で使用されている Rust バインディング

GNOME

  • gjs - GNOME への JavaScript バインディング(概して、GObject のイントロスペクションメカニズムを使用している任意のライブラリへの)

Objective Caml

  • http://alain.frisch.fr/soft.html#spider - OCaml アプリケーションに SpiderMonkey を埋め込むためのバインディング

Perl

  • http://jspl.msg.mx/ - SpiderMonkey と Perl を相互に埋め込むためのバインディング。JavaScript から CPAN モジュールを使用できるようにする JS シェルが含まれています。

Python

  • http://pypi.python.org/pypi/python-spidermonkey

wxWidgets

  • GLUEscript (formerly wxJavaScript)

Code generators

  • jsapigen - C アプリケーションに SpiderMonkey を埋め込むためのバインディングを生成する

拡張機能

  • http://code.google.com/p/jslibs/ - zlib, SQLite, NSPR, ode, libpng, libjpeg, libffi, (...) SpiderMonkey 用のライブラリ

  • http://www.jsdb.org/ - ファイル、ネットワーク、データベース、圧縮、メールなどのネイティブオブジェクトを含む JS シェル

  • http://code.google.com/p/gpsee/ - SpiderMonkey 用の commonJS プラットフォーム、母国語モジュールの相互運用方法、モジュールなど。既存の埋め込みを補完するように設計された GPSEE は、サンプルの簡単な埋め込み、スクリプトホスト、および Mozilla の JS シェルの拡張バージョンを同梱しています。バンドルされているモジュールには、ByteArray、ByteString、CGI、POSIX Signals、FFI などのサポートが含まれています。

  • http://javalikescript.free.fr/ - 拡張可能なツーリングフレームワークを提供することを目的とした汎用 JavaScript プラットフォーム

  • http://www.synchro.net/ - telnet/SSH 掲示板システム (BBS) とインターネットサーバ (HTTP/SMTP/FTP)。JavaScript で記述された追加のインターネットサービス(IRC、IMAP、Gopher など)。

  • https://github.com/mindbit/libjssql/wiki - JDBC にできるだけ似せた、JavaScript 用の汎用 SQL API を提供することを目的としたライブラリ

その他

  • ServerJS, 細分化されたサーバサイドの Javascript の世界に対する標準化の取り組み
  • 0 A.D., JavaScript スクリプトを使った歴史的な RTS ゲーム
  • http://soubok.googlepages.com/javascript
  • Malzilla - SpiderMonkey を使用して、Web ページ上の JavaScript を難読化してマルウェアを除去するためのツール
