RustはMozillaとコミュニティによって作成された、オープンソースのプログラミング言語です。開発者が最新のマルチコアプロセッサの強力な機能を最大限に活用して、高速で安全なアプリケーションを作成できるように設計されたおり、セグメンテーション違反を防ぎ、スレッドの安全性を保証します。これらはすべて習得しやすい構文です。

Rustは最小のランタイムサイズで、ゼロコスト抽象化、移動セマンティクス、保証されたメモリの安全性、データ競合のないスレッド、特性ベースの総称、パターンマッチング、型推論、および効率的なCバインディングを提供します。

Rustを学ぶために以下のことができます。

  • Rustが提供するパワーと利点の詳細については、以下のビデオをご覧ください。
  • オンラインでRust Programming Languageを読んでください。
  • Rustの公式WebサイトでRustコンパイラーをダウンロードし、例を調べて何ができるかを学んでください。
