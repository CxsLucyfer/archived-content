« Gecko Roles ページ

何かへのリンクを表します。このオブジェクトはテキストやグラフィックで表現されますが、ボタンのように動作します。

インターフェイス

  • nsIAccessible
  • nsIAccessibleText
  • nsIAccessibleHyperText
  • nsIAccessibleHyperLink
  • nsIAccessNode

別名

  • AT-SPI: ROLE_LINK
  • ATK: ATK_ROLE_LINK
  • UA: @"AXLink"
  • MSAA/IA2: ROLE_SYSTEM_LINK

使用する要素

  • ARIA: link
  • XUL: <label class="text-link"/>
  • HTML: <a>, <area>
  • XForms: <trigger appearance="minimal"/>.
