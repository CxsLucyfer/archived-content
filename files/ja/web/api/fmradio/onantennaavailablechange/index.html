概要

antennaavailablechange イベントを受け取るイベントリスナ関数を指定します。このイベントは、端末にアンテナが接続された時、または外された時に発生します。

現在のアンテナの状態を確認するには、FMRadio.antennaAvailable プロパティを使用してください。

構文

navigator.mozFMRadio.onantennaavailablechange = funcRef

funcRef の部分は、antennaavailablechange イベントが発生した時に呼び出される関数です。
