概要

frequencychange イベントを受け取るイベントリスナ関数を指定します。このイベントは、ラジオの周波数が変更されるたびに発生します。

構文

navigator.mozFMRadio.onfrequencychange = funcRef

funcRef の部分は、frequencychange イベントが発生した時に呼び出される関数です。

仕様書

仕様はありません。
