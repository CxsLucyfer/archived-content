概要

enabled イベントを受け取るイベントリスナ関数を指定します。このイベントは、端末の FM ラジオが ON になると発生します。

構文

navigator.mozFMRadio.onenabled = funcRef

funcRef の部分は、enabled イベントが発生した時に呼び出される関数です。

仕様書

仕様はありません。
