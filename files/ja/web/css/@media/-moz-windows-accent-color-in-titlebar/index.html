非標準
この機能は標準ではなく、標準化の予定もありません。公開されているウェブサイトには使用しないでください。ユーザーによっては使用できないことがあります。実装ごとに大きな差があることもあり、将来は振る舞いが変わるかもしれません。

メモ: Firefox 58 以降、このメディア特性はウェブコンテンツでは利用できなくなりました。内部 (e.g. to XUL / chrome code) でのみ利用できます。詳しくは バグ 1396066 をご覧ください。

CSS の -moz-windows-accent-color-in-titlebar メディア特性は Gecko 独自のもので、 Microsoft Windows のタイトルバーで強調色が有効であるかどうかでスタイルを適用するために使用することができます。

構文

<integer>
最新の Windows のバージョンでは、ウィンドウのタイトルバーで強調色が有効であれば、これは 1 です。それ以外は 0 です。

メディア: media/visual
min/max 接頭辞の受付: いいえ

例

@media (-moz-windows-accent-color-in-titlebar: 1) {
  h1 {
    color: -moz-win-accentcolortext;
  }

  body {
    background-color: -moz-win-accentcolor;
  }
}

関連情報

  • この特性は、 CSS から Windows 強調色及び重複テキスト強調色にアクセスできるという点で、 Mozilla 独自の <color> 値である -moz-win-accentcolor 及び -moz-win-accentcolortext と深く関係しています。
