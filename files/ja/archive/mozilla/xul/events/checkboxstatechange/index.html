CheckboxStateChange イベントは、<checkbox> 要素の状態が変更されたときに実行されます。

このイベントは、主にアクセシビリティの目的で使用されます。

一般情報

仕様
XUL
インターフェイス
Event
バブル
Yes
キャンセル可能
Yes
ターゲット
element
既定のアクション
None

プロパティ

プロパティ タイプ 説明
target 読取専用 EventTarget イベントのターゲット (DOMツリーの一番上のターゲット)
type 読取専用 DOMString イベントのタイプ
bubbles 読取専用 boolean イベントは通常バブルか
cancelable 読取専用 boolean イベントをキャンセルすることは可能か
