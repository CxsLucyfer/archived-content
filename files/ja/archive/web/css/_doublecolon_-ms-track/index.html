非標準
この機能は標準ではなく、標準化の予定もありません。公開されているウェブサイトには使用しないでください。ユーザーによっては使用できないことがあります。実装ごとに大きな差があることもあり、将来は振る舞いが変わるかもしれません。

::-ms-track CSS 疑似要素は、スライダーコントロールのトラックを表す Microsoft 拡張機能 です。スライダーコントロールは、 <input type="range"> の1つの可能な表現です。

::-ms-track セレクター内で color プロパティを適用すると、トラック領域内の目盛りに影響します。

::-ms-ticks-after 、 ::-ms-ticks-before そして ::-ms-track 疑似要素を共に使用できます。これを行うと、3セットの目盛りが作成されますが、最高のユーザーエクスペリエンスは1セットです。目盛りを完全に削除するには、 color プロパティを transparent に設定します。
