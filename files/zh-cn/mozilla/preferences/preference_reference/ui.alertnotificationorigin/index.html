ui.alertNotificationOrigin 控制由 nsIAlertsService 生成的弹出提示窗口向外滚动时的位置和方向.

  • <dfn style="display:table-cell;padding: 0 5px;border-bottom: none;cursor:inherit;">值:</dfn>integer
  • <dfn style="display:table-cell;padding: 0 5px;border-bottom: none;cursor:inherit;">默认值:</dfn>依据任务栏的位置而定
  • <dfn style="display:table-cell;padding: 0 5px;border-bottom: none;cursor:inherit;">默认已存在:</dfn> no
  • <dfn style="display:table-cell;padding: 0 5px;border-bottom: none;cursor:inherit;">支持版本:</dfn> Gecko 1.8.1.2 (Firefox 2.0.0.2 / Thunderbird 2.0.0.4 / SeaMonkey 1.1)
  • <dfn style="display:table-cell;padding: 0 5px;border-bottom: none;cursor:inherit;">状态:</dfn> Active; last updated 2012-02-22
  • <dfn style="display:table-cell;padding: 0 5px;border-bottom: none;cursor:inherit;">介绍:</dfn> Pushed to Nightly on 2007-01-04
  • <dfn style="display:table-cell;padding: 0 5px;border-bottom: none;cursor:inherit;">Bugs:</dfn> bug 133527

值

0
右下角,由下向上滑出
1
右下角, 由右向左滑出
2
左下角, 由下向上滑出
3
左下角, 由左向右滑出
4
右上角, 由上向下滑出
5
右上角, 由右向左滑出
6
左上角, 由上向下滑出
7
左上角, 由左向右滑出
