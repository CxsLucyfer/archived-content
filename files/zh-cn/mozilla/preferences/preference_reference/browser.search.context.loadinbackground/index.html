--- title: browser.search.context.loadInBackground slug: >- Mozilla/Preferences/Preference_reference/browser.search.context.loadInBackground translation_of: >- Mozilla/Preferences/Preference_reference/browser.search.context.loadInBackground ---

browser.search.context.loadInBackground 控制右键菜单中的"通过<搜索引擎>搜索<选中文字>"打开新标签页后,立即激活该标签页,还是保持当前标签页状态不变.

  • <dfn style="display:table-cell;padding: 0 5px;border-bottom: none;cursor:inherit;">类型:</dfn>boolean
  • <dfn style="display:table-cell;padding: 0 5px;border-bottom: none;cursor:inherit;">默认值:</dfn>false
  • <dfn style="display:table-cell;padding: 0 5px;border-bottom: none;cursor:inherit;">默认已存在:</dfn> yes
  • <dfn style="display:table-cell;padding: 0 5px;border-bottom: none;cursor:inherit;">支持版本:</dfn> Firefox 13.0
  • <dfn style="display:table-cell;padding: 0 5px;border-bottom: none;cursor:inherit;">状态:</dfn> Active; last updated 2012-02-17
  • <dfn style="display:table-cell;padding: 0 5px;border-bottom: none;cursor:inherit;">介绍:</dfn> Pushed to Nightly on 2012-02-15
  • <dfn style="display:table-cell;padding: 0 5px;border-bottom: none;cursor:inherit;">Bugs:</dfn> bug 727131

值

true
搜索页面在后台打开,保持当前标签页不变
false (默认)
搜索页面在前台打开(打开并激活).
