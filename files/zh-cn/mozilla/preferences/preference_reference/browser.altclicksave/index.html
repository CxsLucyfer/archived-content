 browser.altClickSave 控制是否在按住alt键的同时点击链接执行下载动作.

  • 类型:boolean
  • 默认值:  false
  • 默认已存在: yes
  • 支持版本: Gecko 13.0 (Firefox 13.0 / Thunderbird 13.0 / SeaMonkey 2.10)
  • 状态: Active; last updated 2012-03-19
  • 介绍: Pushed to Nightly on 2012-03-02
  • Bugs: bug 713052

值

true
在按住alt键的同时点击链接执行下载该链接的动作.
false (默认)
无下载动作 (查看bug 736985).
