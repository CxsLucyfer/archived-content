已废弃 JSAPI 42
This feature is obsolete. Although it may still work in some browsers, its use is discouraged since it could be removed at any time. Try to avoid using it.

Cast a C integer to a boolean JS::Value without any type checking or error handling.

Please use JS::BooleanValue/JS::TrueValue/JS::FalseValue instead in SpiderMonkey 45 or later.

Syntax

jsval
BOOLEAN_TO_JSVAL(bool b);
Name Type Description
b bool C integer value to be converted to a boolean jsval.

Description

BOOLEAN_TO_JSVAL converts a bool argument, b, to a boolean jsval.

If b is false, the result is JSVAL_FALSE.

If b is true, the result is JSVAL_TRUE.
