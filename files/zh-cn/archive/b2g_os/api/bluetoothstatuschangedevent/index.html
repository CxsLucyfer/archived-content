概要

BluetoothStatusChangedEvent API可以访问有关蓝牙设备状态的任何更改的信息。

当触发以下某个事件时，会发生状态更改:

  • a2dpstatuschange : 当A2DP连接状态发生变化时。看到BluetoothAdapter.ona2dpstatuschanged 了解更多信息。
  • hfpstatuschange : 发生在HFP连接状态发生变化时。看到BluetoothAdapter.onhfpstatuschanged 了解更多信息.
  • pairedstatuschange : 当设备的配对状态发生变化时。看到BluetoothAdapter.onpairedstatuschanged for more information.了解更多信息
  • scostatuschange : 它发生在SCO连接状态改变时。看到BluetoothAdapter.onscostatuschanged for more information.

接口概述

interface BluetoothStatusChangedEvent: Event
{
  readonly attribute DOMString address;
  readonly attribute boolean status;
};

属性

BluetoothStatusChangedEvent.address 只读
表示蓝牙微网中状态发生变化的设备地址的字符串。.
BluetoothStatusChangedEvent.status 只读
表示连接当前状态的布尔值。它可以被启用（true）或禁用（false）。

方法

目前没有。
