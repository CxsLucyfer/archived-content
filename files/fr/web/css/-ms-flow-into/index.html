Non standard
Cette fonctionnalité n'est ni standard, ni en voie de standardisation. Ne l'utilisez pas pour des sites accessibles sur le Web : elle ne fonctionnera pas pour tout utilisateur. Il peut également y avoir d'importantes incompatibilités entre les implémentations et son comportement peut être modifié dans le futur.

La propriété CSS -ms-flow-into est une propriété spécifique de Microsoft extension qui permet d'obtenir ou de définir une valeur identifiant un conteneur <iframe> du document et qui sert de source de données à la région.

Syntaxe

La propriété -ms-flow-into est définie grâce à l'une des valeurs suivantes.

Valeurs

none

La valeur par défaut. Aucun conteneur n'est indiqué.

<custom-ident>

Le nom du conteneur de région.

Syntaxe formelle

[ none | <custom-ident> ]#

Spécifications

Cette propriété ne fait partie d'aucune spécification.
