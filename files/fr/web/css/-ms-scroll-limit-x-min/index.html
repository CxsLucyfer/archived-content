Non standard
Cette fonctionnalité n'est ni standard, ni en voie de standardisation. Ne l'utilisez pas pour des sites accessibles sur le Web : elle ne fonctionnera pas pour tout utilisateur. Il peut également y avoir d'importantes incompatibilités entre les implémentations et son comportement peut être modifié dans le futur.

La propriété -ms-scroll-limit-x-min est une propriété spécifique à Microsoft qui indique la valeur minimale pour la propriété Element.scrollLeft.

Syntaxe

La propriété -ms-scroll-limit-x-min est définie avec une longueur.

Valeurs

length

La valeur minimale pour la propriété scrollLeft. Si la valeur fournie est négative, c'est 0 qui sera utilisé.

Syntaxe formelle

<length>

Spécifications

Cette propriété est une propriété spécifique et ne fait partie d'aucune spécification.

Valeur initiale0
Applicabilitéles éléments de bloc non remplacés et les éléments en bloc en ligne et en bloc (inline-block)
Héritéenon
Médiainteractif
Valeur calculéecomme spécifié
Type d'animationdiscrète
Ordre canoniquel'ordre unique et non-ambigu défini par la grammaire formelle

Notes

Cette propriété n'a aucun effet sur les éléments qui ne défilent pas.

Le comportement choisi avec -ms-scroll-limit-x-min s'appliquera uniquement avec mscontentzoomfactor qui vaut 1 ; pour les autres facteurs de zoom, le comportement de la propriété n'est pas défini.

Cette propriété est uniquement disponible à partir de Windows 8.
