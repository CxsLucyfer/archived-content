Non standard
Cette fonctionnalité n'est ni standard, ni en voie de standardisation. Ne l'utilisez pas pour des sites accessibles sur le Web : elle ne fonctionnera pas pour tout utilisateur. Il peut également y avoir d'importantes incompatibilités entre les implémentations et son comportement peut être modifié dans le futur.

La propriété CSS -ms-scroll-limit est une propriété raccourcie spécifique à Microsoft qui définit les valeurs de -ms-scroll-limit-x-min, -ms-scroll-limit-y-min, -ms-scroll-limit-x-max et -ms-scroll-limit-y-max.

Syntaxe

La propriété -ms-scroll-limit est définie avec une ou plusieurs valeurs de limite de défilmeent dans l'ordre et séparées par des espaces.

Valeurs

-ms-scroll-limit-x-min

La valeur utilisée pour la propriété -ms-scroll-limit-x-min.

-ms-scroll-limit-y-min

La valeur utilisée pour la propriété -ms-scroll-limit-y-min.

-ms-scroll-limit-x-max

La valeur utilisée pour la propriété -ms-scroll-limit-x-max.

-ms-scroll-limit-y-max

La valeur utilisée pour la propriété -ms-scroll-limit-y-max.

Syntaxe formelle

<'-ms-scroll-limit-x-min'> <'-ms-scroll-limit-y-min'> <'-ms-scroll-limit-x-max'> <'-ms-scroll-limit-y-max'>

Spécifications

Cette propriété est une propriété spécifique à Microsoft et ne fait partie d'aucune spécification.

Valeur initialepour chaque propriété individuelle de la propriété raccourcie :
  • -ms-scroll-limit-x-min: 0
  • -ms-scroll-limit-y-min: 0
  • -ms-scroll-limit-x-max: auto
  • -ms-scroll-limit-y-max: auto
Applicabilitéles éléments de bloc non remplacés et les éléments en bloc en ligne et en bloc (inline-block)

Notes

Cette propriété n'a aucun effet sur les éléments qui ne défilent pas.

Le comportement choisi avec -ms-scroll-limit s'appliquera uniquement avec mscontentzoomfactor qui vaut 1 ; pour les autres facteurs de zoom, le comportement de la propriété n'est pas défini.

Cette propriété est uniquement disponible à partir de Windows 8.
