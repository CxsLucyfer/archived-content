Non standard
Cette fonctionnalité n'est ni standard, ni en voie de standardisation. Ne l'utilisez pas pour des sites accessibles sur le Web : elle ne fonctionnera pas pour tout utilisateur. Il peut également y avoir d'importantes incompatibilités entre les implémentations et son comportement peut être modifié dans le futur.

Cette API est disponible sur Firefox OS pour des applications internes seulement.

La valeur de la propriété enabled indique si le wifi est activé ( true ) ou désactivé (false).

Note : Activer ou désactiver le WiFi ne peut être fait en utilisant les paramètres API pour changer le wifi.enabled réglage. Chaque fois que ce réglage change, l'objet WifiManager enverra un événement enabled ou disabled Ces événements peuvent être traitées en utilisant les gestionnaires d'événements WifiManager.onenabled et WifiManager.ondisabled.

Syntaxe

var enabled = navigator.mozWifiManager.enabled

Valeur

Retourne un booléen.

Spécifications

Ne fait partie d'aucune spécification.
