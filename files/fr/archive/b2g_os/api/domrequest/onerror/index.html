Non standard
Cette fonctionnalité n'est ni standard, ni en voie de standardisation. Ne l'utilisez pas pour des sites accessibles sur le Web : elle ne fonctionnera pas pour tout utilisateur. Il peut également y avoir d'importantes incompatibilités entre les implémentations et son comportement peut être modifié dans le futur.

Résumé

Cette propriété renseigne la fonction de callback à appeler quand un DOMRequest échoue.

Syntaxe

requete.onerror = fonction;

Où requete est un objet DOMRequest et fonction est une fonction JavaScript à éxecuter quand la requête ne parvient pas à se terminer sans problème.

Spécification

Ne fait actuellement partie d'aucune spécification.
