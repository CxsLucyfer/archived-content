L'API WebSMS permet à du contenu web de créer, envoyer ou recevoir des SMS (Short Message Services).

Cette API est disponible en utilisant window.navigator.mozSms qui renvoie un objet SmsManager. Voir ci-après pour une liste complète des interfaces :

Interfaces DOM

  • window.navigator.mozSms
  • SmsManager
  • SmsRequest
  • SmsMessage
  • SmsEvent
  • SmsFilter
  • SmsCursor

Interfaces

  • nsISmsDatabaseService
  • nsISmsRequestManager
  • nsISmsService

Compatibilité des navigateurs

Pour des raisons évidentes, cette API est surtout destinée aux navigateurs mobiles.

Fonctionnalité Android Firefox Mobile (Gecko) IE Mobile Opera Mobile Safari Mobile
Support simple Pas de support 12.0 (12.0) Pas de support Pas de support Pas de support

 

Réglages et disponibilité

  • WebSMS est désactivé par défaut et peut être activé en passant le paramètre dom.sms.enabled à true.
  • Une liste blanche comportant les hôtes autorisés à utiliser l'API WebSMS doit être définie en utilisant le paramètre dom.sms.whitelist La valeur par défaut est la chaîne de caractères vide.
  • WebSMS est seulement disponible pour les applications certifiées sur Firefox OS (B2G)
