Не стандартно
Эта возможность не является стандартной и стандартизировать её пока никто не собирается. Не используйте её на сайтах, смотрящих во внешний мир: она будет работать не у всех пользователей. Также могут присутствовать большие несовместимости между реализациями и её поведение может в будущем измениться.

Объект navigator.mozAudioChannelManager предоставляющий доступ к интерфейсу  mozAudioChannelManager который используется для  управления аудио каналами вашего устроства Firefox OS включая настройки эффекта громкости канала, когда кнопки регулировки громкости нажимаются внутри конкретного приложения.

Синтаксис

var mgr = window.navigator.mozAudioChannelManager;

Возвращаемое значение

A mozAudioChannelManager object.

Спецификация

The AudioChannels API has no official spec at the moment; see https://wiki.mozilla.org/WebAPI/AudioChannels for implementation details, WebIDL, etc.

Совместимость с браузерами

Supported in Firefox OS 1.0.1.
