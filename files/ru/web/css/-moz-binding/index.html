Не стандартно
Эта возможность не является стандартной и стандартизировать её пока никто не собирается. Не используйте её на сайтах, смотрящих во внешний мир: она будет работать не у всех пользователей. Также могут присутствовать большие несовместимости между реализациями и её поведение может в будущем измениться.

Устарело Gecko 57 (Firefox 57 / Thunderbird 57 / SeaMonkey 2.54)
Эта возможность была удалена из веб-стандартов. Хотя некоторые браузеры по-прежнему могут поддерживать её, она находится в процессе удаления. Не используйте её ни в старых, ни в новых проектах. Страницы или веб-приложения, использующие её, могут в любой момент сломаться.

The -moz-binding CSS property is used by Mozilla-based applications to attach an XBL binding to a DOM element.

Начальное значениеnone
Применяется квсе элементы, кроме сгенерированного контента и псевдоэлементов
Наследуетсянет
Отображениевизуальный
Обработка значениякак указано
Animation typediscrete
Канонический порядокуникальный неоднозначный порядок, определённый формальной грамматикой

Syntax

/* <url> value */
-moz-binding: url(http://www.example.org/xbl/htmlBindings.xml#checkbox);

/* Global values */
-moz-binding: inherited;
-moz-binding: initial;
-moz-binding: unset;

Values

<url>
The URL for the XBL binding (including the fragment identifier).
none
No XBL binding is applied to the element.

Formal syntax

<url> | none

Example

.exampleone {
  -moz-binding: url(http://www.example.org/xbl/htmlBindings.xml#radiobutton);
}

Specifications
