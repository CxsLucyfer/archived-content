Вышла из употребления
Эта возможность вышла из употребления. Хотя она может продолжать работать в некоторых браузерах, её использование не рекомендуется, поскольку она может быть удалена в любое время. Старайтесь избегать её использования.

Сводка

Метод Object.eval() использовался для вычисления строки JavaScript-кода в контексте объекта, однако, был удалён.

Синтаксис

obj.eval(string)

Параметры

string
Любая строка, представляющая выражение, инструкцию или последовательность инструкций JavaScript. Выражение может содержать переменые и свойства существующих объектов.

Описание

Метод eval больше не используется как метод объекта. Вместо него используйте глобальную функцию eval().

Спецификации

Не является частью какой-либо спецификации.
