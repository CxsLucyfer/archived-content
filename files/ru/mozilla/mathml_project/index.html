Проект MathML сообщества Mozilla представляет собой проект Mozilla по созданию и расширению поддержки MathML в Firefox и других приложениях на базе Mozilla. Смотрите краткий обзор слайдов для инновационных ярмарок на саммите Mozilla 2013.

Обновления

  • Статусы элементов и атрибутов
  • Результаты тестирования MathML 3
  • Неофициальные сборки с применением патчей MathML поддерживается Биллом Гианополисом (Bill Gianopoulos)
  • Больше обновлений и архивов контента

Примеры документов MathML

  • Скриншоты
  • Стартовая страница MathML - с переводами на разные языки ( Арабский, Китайский, Иврит, Тайский)
  • Основы MathML - документ, предназначенный для правильного отображения с использованием только шрифта Symbol, который предустановлен по умолчанию в большинстве конфигураций ОС.
  • MathML Torture Test - сравнительное тестирование рендеринга MathML и TeX.
  • Демо некоторых тегов MathML: mfrac, mo, mtable, mspace, mmultiscripts, msqrt-mroot.
  • MathML Extras - технологическая демонстрация некоторых интересных результатов, использующих браузерное окружение.
  • Разнообразные тесты MathML - эти страницы были импорированы из layout/mathml/tests/.
  • Блог с комментариями, которые включают MathML.
