Podstawy Web Services
Krótkie wprowadzenie do Web Services.
Web service to kolekcja protokołów i standardów używanych do wymiany danych między aplikacjami lub systemami. Aplikacje napisane w różnych językach programistycznych i uruchomione na różnych platformach mogą korzystając z web services wymieniać dane poprzez sieć taką jak Internet w sposób podobny do tego jaki wykorzystują do komunikacji wewnątrz jednego komputera. W przypadku XML-owego web service, wszystkie wymieniane dane są formatowane w XML-u. [1]
