This page aims to give an overview of what version of Lightning is compatible with what version of Thunderbird. It also shows which Mozilla Platform Version is used in the respective version of Sunbird and Lightning.

From Thunderbird 74 onwards Lightning does not exist as a separate add-on but is integrated into Thunderbird.

From Thunderbird 68 onwards Lightning has the same version number as the corresponding Thunderbird.

Note: If you are having trouble and calendar is missing, first see calendar update issues, because after Thunderbird 52 Lightning is built-in (bundled with Thunderbird) and you should not need to download the extension. But if you are running Thunderbird version 45 or older use a download link below.

Stable ESR Releases

Thunderbird 10 and 17 were considered Extended Support Releases (ESR), which Lightning is compatible with. We followed the ESR policy of Thunderbird and only did releases for security fixes. Additionally, some fixes may have been backported for important regressions related to enterprise features like invitations.
