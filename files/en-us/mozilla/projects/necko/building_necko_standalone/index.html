Obsolete since Gecko 9.0 (Firefox 9.0 / Thunderbird 9.0 / SeaMonkey 2.6)
This feature is obsolete. Although it may still work in some browsers, its use is discouraged since it could be removed at any time. Try to avoid using it.

You can build the Necko networking engine by itself quite simply, by specifying netwerk as the value of the --enable-application compile option.

Note: Support for building Necko standalone was removed in GeckoÂ 9.0.

An example .mozconfig file

ac_add_options --enable-application=netwerk
