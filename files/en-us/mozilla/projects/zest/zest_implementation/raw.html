The first version of Zest has been released but is intentionally very basic. Future versions of Zest are planned which will significantly increase the scope of the language.

The first version

  • Is aimed at creating scripts for reproducing basic security vulnerabilities
  • Includes a Java reference implementation, which conforms to JSR 223
  • Has been included in a proof-of-concept OWASP ZAP add-on

The next version is underdevelopment - more details soon.
