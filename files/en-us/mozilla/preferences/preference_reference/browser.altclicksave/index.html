The preference browser.altClickSave controls whether clicking a link while holding the alt key starts the download of that link.

  • Type:boolean
  • Default value:  false
  • Exists by default: yes
  • Application support:Firefox 13.0
  • Status: Active; last updated 2012-03-19
  • Introduction: Pushed to Nightly on 2012-03-02
  • Bugs: bug 713052

Values

true
Clicking a link while holding the alt key starts the download of that link.
false (default)
No action triggered (see bug 736985).
