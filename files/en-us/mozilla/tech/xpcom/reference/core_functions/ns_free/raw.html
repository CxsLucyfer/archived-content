Summary

Frees a block of memory using the XPCOM memory manager. 

#include "nsXPCOM.h"

void NS_Free(
  void* aPtr
);

Parameters

aPtr
[in] A pointer to the block of memory to free.

Remarks

This function provides a convenient way to access the XPCOM memory manager. It is equivalent to calling free on the nsIMemory instance returned from NS_GetMemoryManager.

History

This function was finalized for Mozilla 1.8. See bug 267767 for details.
