The boxObject belonging to a XUL browser, editor or iframe element implements this interface.

Attributes

Attribute Type Description
docShell nsIDocShell Read only. The nsIDocShell object that manages the contained document. The browser.xml, editor.xml and general.xml bindings use this property to gain access to other properties such as webNavigation, contentDocument and contentWindow.
