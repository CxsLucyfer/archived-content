nsIXFormsNSModelElement is implemented by all XForms <model/> elements. It defines Mozilla extensions to the nsIXFormsModelElement interface.

Methods

getInstanceDocuments

 nsIDOMNodeList getInstanceDocuments();

getInstanceDocuments returns a nsIDOMNodeList containing all the instance documents for the model, making it possible to enumerate over instances in the model without knowing their names.
