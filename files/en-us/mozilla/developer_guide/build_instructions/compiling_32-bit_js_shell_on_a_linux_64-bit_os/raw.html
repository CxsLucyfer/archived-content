Normally, a Javascript shell is built as follows (assuming an Ubuntu installation):

cd js/src
autoconf2.13
mkdir obj
cd obj
../configure --enable-debug --disable-optimize
make

To build a 32-bit shell, the following commands will work:

cd js/src
autoconf2.13
mkdir obj
cd obj
CFLAGS="-m32" CXXFLAGS="-m32" linux32 ../configure --enable-debug --disable-optimize
make
