This article provides a brief description of how to use the keyboard to simulate TV remote control buttons.

We use the keyboard to simulate TV remote control buttons when developing apps for Firefox OS on TV. The following explains the keyboard equivalents to the TV remote control buttons in the three different platforms: Mac OS X, Windows and Linux.

Mac OS X

Remote Control Keyboard
home fn + left
back backspace
enter return
option fn + ctrl + I

Windows

Remote Control Keyboard
home home
back backspace
enter enter

Linux

Remote Control Keyboard
home home
back backspace
enter enter
option shift + F10
