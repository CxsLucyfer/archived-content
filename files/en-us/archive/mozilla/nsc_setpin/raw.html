Name

NSC_SetPIN - Modify the user's PIN.

Syntax 

CK_RV NSC_SetPIN(
  CK_SESSION_HANDLE hSession,
  CK_CHAR_PTR pOldPin,
  CK_ULONG ulOldLen,
  CK_CHAR_PTR pNewPin,
  CK_ULONG ulNewLen
);

Parameters

NSC_SetPIN takes five parameters:

hSession
[Input]
pOldPin
[Input] .
ulOldLen
[Input] .;
pNewPin
[Input] .;
ulNewLen
[Input] .

Description

NSC_SetPIN modifies the PIN of user that is currently logged in. This is only valid for the PRIVATE_KEY_SLOT.

Return value

CKR_OK

Examples 

#include <assert.h>
