Summary

Sets the value of a key, when that value is a number.

Method of

WinReg object

Syntax

int setValueNumber ( String subkey, String valname, Number value );

Parameters

The method has the following parameters:

subkey
The key path to the appropriate location in the key hierarchy, such as "Software\\Netscape\\Navigator\\Mail".
valname
The name of the value-name/value pair whose value you want to change.
value
A number representing the new string value.

Returns

0 if it succeeded; a nonzero number if it failed to schedule the action. For a list of possible values, see Return Codes.

Description

The setValueNumber method sets the value of a key when that value is a number. Use this method if the value you want to set is a number. If the value is not a number, use the setValue or setValueString methods instead.
