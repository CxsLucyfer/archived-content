BlackConnect was a Blackwood subproject, trying to build a Java-to-XPCOM bridge to allow interoperability between Java and XPCOM components. The goal was to be able to implement XPCOM components written in Java instead of C++.

BlackConnect is now superseeded by JavaXPCOM. For the remainders of the BlackConnect project, see here.
