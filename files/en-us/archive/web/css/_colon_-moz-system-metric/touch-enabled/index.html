Non-standard
This feature is non-standard and is not on a standards track. Do not use it on production sites facing the Web: it will not work for every user. There may also be large incompatibilities between implementations and the behavior may change in the future.

The :-moz-system-metric(touch-enabled) CSS pseudo-class will match an element if the device on which the content is being rendered offers a supported touch-screen interface.

Note: Since Firefox 58, this pseudo-class is no longer available to web content — it is only available internally (e.g. to XUL / chrome code). See bug 1396066 for more details. Use the -moz-touch-enabled media query instead.
