Introduction

Serves to show/change multiline text (see the spec).

Attributes

  • UI Common
    • appearance - isn't supported.
    • accesskey - used to specify the keyboard shortcut for focusing this control.
  • Single-Node Binding
  • Special
    • inputmode - isn't supported.
    • incremental - supported. The default value is false.


Type restrictions

The textarea element can be bound to a node of type xsd:string or any type derived from it.

Representations

The XForms textarea element is represented by a multiline text field (xhtml/xul).

Characteristics

  • analogous widgets are <xhtml:textarea/> and <xul:textbox multiline="true"/>
  • if the incremental attribute has the value true, then the bound instance node is updated on every user input.
