Description

The footer should be a small, unobtrusive strip along the bottom of every MDC page, made up mostly of text with links to more detailed information about the "fine print" for MDC.

Block Layout

Footer

Along the bottom of all MDC pages should be footer strip that contains the following elements:

  • Last update time for the page
  • Copyright & license information
  • Disclaimer information
  • Link to "About MDC" pages
