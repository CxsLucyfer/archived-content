Description

The page tools area provides access to the MediaWiki tools used to manage the wiki pages. The design should be subtle, and not call too much attention to itself. The suggested design is that there be a single header and then a just list of hyperlinks.

Block Layout


Page Tools

When a user isn't logged in, the available tools will be:

  • Edit
  • History

When a user is logged in, the available tools will be:

  • Edit
  • History
  • Source
  • Watch
  • Move
  • Delete
  • Referenced By (MediaWiki's "What links here"
  • Related Changes

seperated from the above tools are some extra wiki functions which are only available when one is logged in:

  • Upload
  • Special pages
