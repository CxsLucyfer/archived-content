Description

The main Webwatch Index Page displays the last 5 news items for each topic, and provides links to news index by topic pages (which use the same block layout) which display all news items for a specific topic.

In all cases, the entries are taken from RSS feeds from the MDC DevNews weblog.

Block Layout

Common Elements

This page uses the common navigation bar, search box, page tools and footer, but a custom side bar.

Side Bar Content

The side bar for the News Index Page will provide links to date-based archive pages, listed by month. Each of these index pages will use the same block layout as the main index page, but will feature all news items from an entire month, sorted by topic.
