Description

The Community Index Page displays all links to community resources (irc channels, mailing lists, newsgroups) sorted by topic.

Block Layout

Common Elements

This page uses the common navigation bar, search box, page tools and footer, but a custom side bar.

Side Bar Content

The side bar for the Community Index Page will be non-standard, and be used to provide help to readers by linking to pages that describe how to leverage the community resources.

  • "How to subscribe to a newsgroup"
  • "How to join an IRC channel"
  • "How to join a mailing list"

Community items sorted by topic

An exhaustive list of all community links:

  • Small headers for each topic name
  • Listings of community links within each topic section in alphabetical order, and sorted by: web forums, newsgroups, mailing lists, irc channels
