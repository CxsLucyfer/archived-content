Description

The MDC home page is a static page that provides the welcome experience for new and returning users. The page will draw together content from both the wiki and the MDC weblogs.

Block Layout

Common Elements

This page uses the common navigation bar, search box, page tools and footer, but a custom side bar.

Sidebar

The sidebar for the MDC Home Page will be used to inform frequent visitors about recent announcements and upcoming events, based on content provided by MDC Devnews and MDC Webwatch.

  • Developer News
    • latest 5 entries from MDC Devnews
    • "More developer news .." link that leads to MDC Devnews, which should have a layout/design that matches MDC (ideally, the user wouldn't be aware that they've left the wiki and entered a blog)
  • Web Watch
    • latest 5 entries from MDC Webwatch
    • "More from web watch .." link that leads to MDC Webwatch, which should have a layout/design that matches MDC (ideally, the user wouldn't be aware that they've left the wiki and entered a blog)
  • Discuss (leads to the discussion page)

Featured Article

The primary focal point on the MDC Home Page is billboard for an article. The content of this space should be updated regularly (weekly? fortnightly?) and highlight the "new hotness" for developers.

  • Combination of graphics and text
  • Contains link to actual article

(see Apple Developer Center for inspiration)

Previously Featured Articles

The last five articles to have appeared as "Featured Articles" (above) will be listed here as links, allowing returning users quick routes to finding the content again.

  • Small header "Recent Features"
  • Previous feature article titles as links
  • "More articles ..." link that leads to article index

Latest Changed or Updated Articles

List of the last five articles to have been changed or updated. (tbd: can we draw this information out of the wiki dynamically?)

Topic & Reference Categories

Alphabetical listing of the various topic & reference pages available on MDC, seperated into "Technology Topics", "Developer Topics" and "Reference Libraries"

  • Small headers for "Technology Topics", "Developer Topics" and "Reference Libraries"
  • Alphabetical listings of topic and reference pages as links
