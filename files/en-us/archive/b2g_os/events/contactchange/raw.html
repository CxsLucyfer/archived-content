The contactchange event is fired each time a contact is added, updated or removed from the device's contact database.

Bubbles No
Cancelable No
Target objects ContactManager
Interface MozContactChangeEvent

Properties

Property Type Description
target (readonly) EventTarget The event target (the topmost target in the DOM tree).
type (readonly) DOMString The type of event.
bubbles (readonly) boolean Does the event normally bubble?
cancelable (readonly) boolean Is it possible to cancel the event?
contactID (readonly) DOMString The unique ID of the contact that has been changed.
reason (readonly) DOMString The reason why the contact has been updated.
