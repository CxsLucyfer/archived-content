The enabled handler is executed when Wifi is enabled on the device.

General info

Specification
Wifi (Firefox OS specific) and WifiDirect (Firefox OS specific)
Interface
Event
Bubbles
No
Cancelable
No
Target
WifiManager and MozWifiP2pManager
Default Action
None

Properties

Unknown

Example

navigator.mozWifiManager.onenabled = function() {
  console.log( "Wifi has been enabled" );
};
