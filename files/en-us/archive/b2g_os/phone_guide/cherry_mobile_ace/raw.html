Device specifications

Software

  • OS: FirefoxOS 1.3T
  • Gecko Version: 28.1
  • Built-in Apps: Calendar, Camera, Clock, Email, File Manager, FM, Gallery, Marketplace, Music, Radio, Settings, Usage, Video.
  • Built-in Manufacturer Apps: Cherry Fun Club, eWarranty.

Hardware

  • Chipset: Spreadtrum SC6821
    • CPU: ARM Cortex-A5
      • Cores: 1
      • Speed: 1Ghz
    • GPU: ARM Mali-400
  • Memory: 128MB
  • Internal storage: 256MB
    • 72.8MB available for applications
    • 8.9MB available for media
  • External storage: MicroSD Card up to 32GB
  • Display:
    • Screen Size: 3.5in
    • Display Technology: TFT, Capacitive
    • Resolution: 320x480 pixels HVGA
    • Pixel Density: 165 PPI
    • Color: Yes
  • Camera:
    • Rear: 2MP, fixed-focus
    • Front: None
  • Battery:
    • Standard Voltage: 3.7V
    • Rating Capacity: 1100mAh/4.07Wh
    • Removable: Yes
    • Model: CM-8P
  • Size: 115 x 62.3 x 12.2 mm
  • Weight: Approx. 108g
  • Audio: Earphones, Loudspeaker, 3.5mm jack
  • USB: Micro USB, USB 2.0
  • SIM: Dual SIM
  • WLAN:
    • Standards: 802.11b/g/n
    • Security: WEP, WPA-PSK, WPA-EAP
  • Bluetooth:4.0
  • Network: GPRS, EDGE, WAP, MMS
  • FM Radio: Yes
  • GPS: Yes
  • Sensors: Accelerometer
