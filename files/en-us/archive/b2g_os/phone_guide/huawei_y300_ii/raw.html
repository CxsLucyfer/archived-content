The Huawei Ascend Y300 II is very similar to the Android-based Ascend Y300 device, except that it runs Firefox OS 1.1 instead of Google Android.

Get a device

The Huawei Ascend Y300 II is available through standard retail channels.

Countries Available

  • Philippines
  • Mexico

Operators​

  • Movistar

Updating & upgrading the software

The device's system image is locked, so you can only update the software when the network operator releases an OTA update. The device system image is based on Firefox OS 1.1.

Device specifications

  • CPU: Qualcomm Snapdragon MSM8225 S4 Play, Dual Core 1GHz, ARMv7 Cortex A5
  • GPU: Qualcomm Adreno 203
  • Camera: 5 megapixel rear with LED Flash, 0.3 megapixel front VGA, Geo Tagging
  • Battery: 1730 mAh
  • Display: 4.0”  16M color display, Multitouch
  • Resolution: WVGA 480x800 pixels, 233 ppi
  • Internal memory: 512MB RAM, 4GB Internal Storage
  • External memory: microSD, microSDHC support up to 32GB
  • Size: 124.5 x 63.8 x 11.2 mm
  • Weight: Approx. 130g

Colors Available

  • Charcoal Black
  • Pearl White
