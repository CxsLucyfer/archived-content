Get a device

The ZTE Open II is available through standard retail channels; for example see the Movistar page.

Operators

  • Movistar

Updating & upgrading the software

The device's system image is locked, so you can only update the software when the network operator releases an OTA update. The device system image is based on Firefox OS 1.2.

Device specifications

  • CPU: Qualcomm MSM8210 Snapdragon 200 dual core 1.2 GHz CPU
  • GPU: Adreno 302
  • Camera: 2 megapixel rear
  • Battery: 1150 mAh
  • Display: 3.5" HVGA 320 x 480 pixels
  • Resolution: 320 x 480 pixels
  • Internal memory: 256 MB, 2GB Internal Storage
  • External memory: microSD, up to 32 GB
  • Size: 118 x 61 x 10.5 mm
  • Weight: Approx. 120 gr.
