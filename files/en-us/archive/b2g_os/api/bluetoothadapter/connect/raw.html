This API is available on Firefox OS for internal applications only.

Summary

The connect method is used to connect the device's adapter to a specific service of a remote device.

Syntax

var request = instanceOfBluetoothAdapter.connect(device[, service]);

Parameters

device
A BluetoothDevice object representing the remote device to connect with.
service Optional
A number representing the UUIDs of the service to connect with, as defined in the Bluetooth specification. If that parameter is undefined, the connection will try to connect to all the possible and compatible services available on the remote device.

Returns

A DOMRequest object to handle the success or error of the operation.

Specification

Not part of any specification yet. It should be discussed as part of the W3C's System Applications Working Group.
