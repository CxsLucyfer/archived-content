This API is available on Firefox OS for internal applications only.

Summary

Specifies an event listener to receive pairedstatuschanged events. Those events occur when a pairing process is completed.

Syntax

instanceOfBluetoothAdapter.onpairedstatuschanged = funcRef

Where funcRef is a function to be called when the pairedstatuschanged event occurs. That function gets a BluetoothStatusChangedEvent object as its first parameter.

Specification

Not part of any specification yet. It should be discussed as part of the W3C's System Applications Working Group.
