This API is available on Firefox OS for internal applications only.

Summary

The devices property gives direct access to all the Bluetooth devices discovered in the area surrounding the device's adapter.

Syntax

var devices = instanceOfBluetoothAdapter.devices

Value

Return an Array of BluetoothDevice objects.

Specification

Not part of any specification yet. It should be discussed as part of the W3C's System Applications Working Group.
