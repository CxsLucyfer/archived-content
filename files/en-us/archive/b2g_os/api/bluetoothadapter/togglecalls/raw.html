This API is available on Firefox OS for internal applications only.

Summary

The toggleCalls method is used to allow HFP devices to switch from one call to another on a CDMA network.

Note: In a GSM network the dialer observes call operations from RIL call state changes; however, in a CDMA network RIL call states do not change under some call operations, so HFP devices need this additional method to have the dialer to be informed of the operation.

Syntax

var request = instanceOfBluetoothAdapter.toggleCalls();

Returns

A DOMRequest object to handle the success or error of the operation.

Specification

Not part of any specification yet. It should be discussed as part of the W3C's System Applications Working Group.
