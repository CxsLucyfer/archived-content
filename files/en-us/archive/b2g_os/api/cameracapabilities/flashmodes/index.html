This API is available on Firefox OS for privileged or certified applications only.

Summary

The flashModes property is an Array of strings identifying each of the flash modes supported by the camera. Possible values are auto, off, on or torch.

Syntax

flash = instanceOfCameraControl.capabilities.flashModes

Return an Array of strings.

Example

var options = {
  camera: navigator.mozCameras.getListOfCameras()[0]
};

function onSuccess(camera) {
  var flash = camera.camera.capabilities.flashModes;

  flash.forEach(function (value) {
    console.log(value)
  });
};

navigator.mozCameras.getCamera(options).then(onSuccess);

Specification

Not part of any specification; however, this API should be removed when the WebRTC Capture and Stream API has been implemented.
