The readNDEF() method of the MozNFCTag interface retrieves the contents of the NDEF stored on the Tag, and returns an array of MozNDEFRecord objects.

Syntax

Promise<sequence<MozNDEFRecord>> readNDEF();

Parameters

None.

Returns

A Promise object which will be resolved as an array of MozNDEFRecord.

Errors

DOMError with error type is InvalidStateError if the tag is already lost.

Specifications

The NFC implementation in Gecko follows the NFC Forum specifications.

Browser compatibility

Supported in Firefox OS 2.0 for certified-apps.
Available in privileged apps as of Firefox OS 2.2.
