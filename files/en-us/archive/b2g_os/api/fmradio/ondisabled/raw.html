    1. FMRadio

Specifies an event listener to receive disabled events. Those events occur when the device's FM radio is turned off.

Syntax

navigator.mozFMRadio.ondisabled = funcRef

Where funcRef is a function to be called when the disabled event occurs.

Specifications

Not part of any specification.

Browser compatibility

Supported in Firefox OS 1.0.
