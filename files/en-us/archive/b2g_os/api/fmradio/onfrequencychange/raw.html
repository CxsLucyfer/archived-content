    1. FMRadio

Specifies an event listener to receive frequencychange events. Those events occur whenever the radio frequency is changed

Syntax

navigator.mozFMRadio.onfrequencychange = funcRef

Where funcRef is a function to be called when the frequencychange event occurs.

Specifications

Not part of any specification.

Browser compatibility

Supported in Firefox OS 1.0.
