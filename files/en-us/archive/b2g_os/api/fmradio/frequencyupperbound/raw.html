    1. FMRadio

This API is available on Firefox or Firefox OS for installed or higher privileged applications.

Summary

The frequencyUpperBound property value indicates the maximum frequency up to which the seek method searches for radio stations.

Syntax

var max = instanceOfFMRadio.frequencyUpperBound

Value

Return a number representing a frequence expressed in MHz (MegaHertz).

Specification

Not part of any specification.
