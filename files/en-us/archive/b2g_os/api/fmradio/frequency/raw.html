    1. FMRadio

This API is available on Firefox or Firefox OS for installed or higher privileged applications.

Summary

The frequency property value returns the current radio frequency.

Syntax

var frequency = instanceOfFMRadio.frequency

Value

Return a number representing the radio frenquence express in MHz (MegaHertz).

Example

window.addEventListener('frequencychange', function () {
  console.log("The frequency has changed to: " + navigator.mozFMRadio.frequency + "MHz");
});

Specification

Not part of any specification.
