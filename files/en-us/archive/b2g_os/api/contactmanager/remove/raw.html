This API is available on Firefox OS for privileged or certified applications only.

Summary

The remove method is used to delete a contact from the device's contact database.

Syntax

var instanceOfDomRequest = window.navigator.mozContacts.remove(contact);

Parameters

contact
A mozContact object representing the contact to remove.

Return value

A DOMRequest object to handle the success or error of the operation.
