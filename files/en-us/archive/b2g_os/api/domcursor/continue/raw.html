The continue method is used to tell the cursor to move to the next result. The DOMCursor's success or error is called with the DOMCursor's result updated accordingly.

Note:Â This feature is available in Web Workers.

Syntax

instanceOfDOMCursor.continue()

Specifications

Not currently part of any specification.

Browser compatibility

Supported in Firefox 21.
Available in workers in Firefox 41.
Support removed in Firefox 61.
