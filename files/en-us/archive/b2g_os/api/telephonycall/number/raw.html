This API is available on Firefox OS for internal applications only.

The number read-only property of the TelephonyCall interface indicates the telephone number associated with the current phone call. Replaced by TelephonyCall.id in Firefox OS 2.0+.

Syntax

var calledNumber = myCall.number;

Value

A DOMString indicating the number associated with the call. This can be either the number dialed for an outgoing call, or the phone number reported by caller ID for an incoming call.

Examples

// Telephony object
var tel = navigator.mozTelephony;

// Place a call
tel.dial("123456789").then(function(call) {
  call.number;
});

// Receive a call
tel.onincoming = function (e) {
  var incomingCall = e.call;

  // Get the number of the incoming call
  console.log(incomingCall.number);

  // Answer the call
  incomingCall.answer();
};
