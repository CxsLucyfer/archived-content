This API is available on Firefox OS for privileged or certified applications only.

The contactID property represents the unique ID of the contact that has been changed.

Syntax

var id = instanceOfMozContactChangeEvent.contactID

Value

Return a string.

Specifications

This feature is currently not defined in any specification.

Browser compatibility

Supported in Firefox OS 1.0.1.
