This API is available on Firefox OS for internal applications only.

Summary

The status property value provides the voicemail status that caused the event to be triggered.

Syntax

var status = instanceOfMozVoicemailEvent.status

Value

Return a MozVoicemailSatus object.

Specification

Not part of any specification yet.
