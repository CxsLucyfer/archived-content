This API is available on Firefox OS for internal applications only.

Summary

The returnNumber property represents the return call number for the voicemail status.

Syntax

var nbr = instanceOfMozVoicemailStatus.returnNumber

Value

Return a string, or null if no return number is provided.

Specification

Not part of any specification yet.
