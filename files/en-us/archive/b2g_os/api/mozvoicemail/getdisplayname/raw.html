This API is available on Firefox OS for internal applications only.

Summary

The getDisplayName method is used to get the display name of the voicemail box dialing number of a specified service.

Syntax

var name = navigator.mozVoicemail.getDisplayName([serviceId]);

Properties

serviceId Optional
A number representing the voicemail service to access.

Returns

A string or null if no number was found.

Specification

Not part of any specification yet.
