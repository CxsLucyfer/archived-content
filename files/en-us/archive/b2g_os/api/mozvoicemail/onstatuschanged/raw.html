This API is available on Firefox OS for internal applications only.

Summary

Specifies an event listener to receive statuschange events. Those events occur when the voicemail status has changed.

Syntax

navigator.mozVoicemail.onstatuschange = funcRef

Where funcRef is a function to be called when the statuschange event occurs. This callback function takes a MozVoicemailEvent object as its first parameter.

Specification

Not part of any specification yet. It should be discussed as part of the W3C's System Applications Working Group.
