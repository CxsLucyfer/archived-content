This API is available on Firefox OS for internal applications only.

The ownAudioChannel() method of the Telephony interface designates the calling app as the owner of the telephony calls from the AudioChannel policy.

Note: The calling app needs to include the audio-channel-telephony permission in its manifest permissions to successfully call ownAudioChannel().

Syntax

myTelephony.ownAudioChannel();

Parameters

None.

Returns

Void.

Examples

// Telephony object
var tel = navigator.mozTelephony;

// Take ownership of telephony calls in the telephony audio channel
tel.ownAudioChannels();
