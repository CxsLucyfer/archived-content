This API is available on Firefox OS for privileged or certified applications only.

Summary

The focusModes property value is a string that defines which kind of focus mode the camera should use. When set, its new value must be chosen from the list of options specified by  CameraCapabilities.focusModes.

Syntax

var focus = instanceOfCameraControl.focusModes

Value

Return a string representing the current focus mode. The default value is auto or fixed (this depends on the device).

Example

var options = {
  camera: navigator.mozCameras.getListOfCameras()[0]
};

function onSuccess( camera ) {
  var capabilities = camera.camera.capabilities;

  if (capabilities.focusModes.indexOf('macro') > -1) {
    capabilities.focusModes = 'macro';
  }
};

navigator.mozCameras.getCamera(options).then(onSuccess)

Specification

Not part of any specification; however, this API should be removed when the WebRTC Capture and Stream API has been implemented.
