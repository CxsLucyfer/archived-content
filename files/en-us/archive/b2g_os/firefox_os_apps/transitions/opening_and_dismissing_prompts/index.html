The animations used to take you between opening and closing/dismissing prompts.

Visualization

Code

/* Launch Value Selector */
animation: fadeIn 0.2s forwards;
@keyframes fadeIn {
  0%   { opacity: 0; }
  100% { opacity: 1; }
}

/* Close Value Selector */
animation: fadeOut 0.2s forwards;
@keyframes fadeOut {
  0%   { opacity: 1; }
  100% { opacity: 0; }
}
