Firefox OS web components

Buttons
Buttons are used to perform explicit actions. Buttons may be text or images.
Checkbox
A checkbox is used to provide a yes/no choice for an option, in the same way as standard HTML checkboxes.
Header
A header is a dedicated space at the top of the screen, most often used to display the view title. It can also include navigation, action buttons, and other controls.
Text input
Text inputs are form fields that allow the user to input text. These include standard single line, multi-line, and PIN/password inputs.
