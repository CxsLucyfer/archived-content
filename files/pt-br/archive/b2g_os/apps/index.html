Aplicações para dispositivos Firefox OS são simplesmente aplicações web abertas; elas são compostas inteiramente por tecnologias web abertas tal como JavaScript, HTML, e CSS. Portanto nossa documentação primaria para aplicações cobre quase tudo que você precisa saber, há alguns documentos específicos apresentados aqui para desenvolvedores e testadores do Firefox OS.
