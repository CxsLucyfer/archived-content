WebTelephony é uma API que possibilita fazer chamadas de voz utilizando a linha telefônica do aparelho a partir de código JavaScript.

A API está acessível utilizando window.navigator.mozTelephony. Veja abaixo os métodos:

Interfaces do DOM

  • window.navigator.mozTelephony
  • Telephony
  • TelephonyCall
  • CallEvent
