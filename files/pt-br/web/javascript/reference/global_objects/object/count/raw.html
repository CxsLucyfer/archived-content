A propriedade __count__ usada para armazenar a contagem de propriedades enumeráveis sobre o objeto, mas ele foi removido.

Sintaxe

obj.__count__

Exemplos

{ 1: 1 }.__count__              // 1
[].__count__                    // 0
[1].__count__                   // 1
[1, /* hole */, 2, 3].__count__ // 3

Especificações

Não faz parte de qualquer especificação.
